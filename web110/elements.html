HTML ELEMENTS

Element Description Block, Inline or N/A Container or Empty
<!-- --> HTML Comments do not render in the browser. They give detail about the instructions and credit. N/A Container
<!DOCTYPE> This HTML declaration tells the browser which version of HTML to expect to render. N/A Container
<a> This tag defines a hyperlink, which is used to link from one page to another. Inline Container
<article> This tag specifies independent, self-contained content. Block Container
<aside> This tag defines some content aside from the content it is placed in. N/A Container
<body> This tag defines the document's body. Block Container
<br> This tag inserts a single line break. Inline Empty
<dd> This tag is used to describe a term/name in a description list. Block Container
<div> This tag defines a division or a section in an HTML document. Block Container
<dl> This tag defines a description list. Block Container
<dt> This tag defines a term/name in a description list. Inline Container
<em> This tag is a phrase tag. It renders as emphasized text. Inline Empty
<fieldset> This tag is used to group related elements in a form. Block Container
<figcaption> This tag defines a caption for a <figure> element. Inline Container
<figure> This tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. Block Container
<footer> This tag defines a footer for a document or section. Block Container
<form> This tag is used to create an HTML form for user input. Block Container
<h1> to <h6> These tags are used to define HTML headings. Inline Container
<head> This element is a container for all the head elements. Block Container
<header> This element represents a container for introductory content or a set of navigational links. Block Container
<html> This tag tells the browser that this is an HTML document. Block Container
<iframe> This tag specifies an inline frame. Inline Container
<img> This tag defines an image in an HTML page. Inline Empty
<input> This tag specifies an input field where the user can enter data. Inline Empty
<label> This tag defines a label for several elements. Inline Container
<li> This tag defines a list item. Inline Container
<link> This tag defines a link between a document and an external resource. Inline Empty
<meta> This tag always goes inside the <head> element, and they provide metadata about the HTML document. Inline Empty
<nav> This tag defines a set of navigation links. Block Container
<ol> This tag defines an ordered list. An ordered list can be numerical or alphabetical. Block Container
<p> This tag defines a paragraph. Inline Container
<script> This tag is used to define a client-side script (JavaScript). Block Container
<section> This tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document. Block Container
<strong> This tag is a phrase tag. It defines important text. Inline Container
<style> This tag is used to define style information for an HTML document. Block Container
<table> This tag defines an HTML table. Block Container
<td> This tag defines a standard cell in an HTML table. Inline Container
<th> This tag defines a header cell in an HTML table. Inline Container
<title> This tag is required in all HTML documents and it defines the title of the document. Inline Container
<tr> This tag defines a row in an HTML table. Block Container
<ul> This tag defines an unordered (bulleted) list. Block Container
